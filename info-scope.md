## ✅ 信息质量标准

### 优先级排序

**最高优先级**（必须关注）：
1. 重大政策变化（影响行业/投资）
2. 技术重大突破（GPT级别）
3. 市场系统性风险（经济危机、战争）
4. 直接影响现有项目的技术更新

**中优先级**（定期查看）：
1. 行业趋势分析
2. 新技术/工具发布
3. 优质公司财报
4. 地缘政治变化

**低优先级**（有空看看）：
1. 技术八卦/花边新闻
2. 个股小道消息
3. 非核心技术的细节更新

---

### 信息可信度评判标准

**我认为可信的信息特征**：
- [x] 有明确的数据来源（官方公告、财报、研究报告）
- [x] 有多个独立信息源验证
- [x] 发布方无明显利益冲突（政府、学术机构 > 自媒体）
- [x] 有历史可追溯性（可以查证过往准确率）
- [x] 逻辑自洽，符合基本常识

**我会自动过滤的信息**：
- [x] 标题党 / 情绪化内容
- [x] 二手三手解读（没有原始来源）
- [x] 匿名爆料（无法验证）
- [x] 明显的营销软文
- [x] "据知情人士透露" 类消息
- [x] 预测性内容（没有数据支撑）

## 🚫 信息边界

**我不需要关注**：
- 娱乐八卦、明星新闻
- 体育赛事（除非涉及商业/技术）
- 游戏行业（非核心）
- 房地产市场（暂时不是决策点）
- 加密货币短期波动（太多噪音）

**原因**：
- 这些信息对当前决策没有帮助
- 容易造成信息过载
- 投入产出比低
